# WeatherDashBoard

I little weather dash board

## setup python
### setup env
```bash
python3 -m venv venv 
```

### activate env
```bash
source ./venv/bin/activate 
```

### install dep
```bash
pip install . 
```

## setup reflex
```bash
reflex init
```

## run dev mode
```bash
reflex run --loglevel debug
```
